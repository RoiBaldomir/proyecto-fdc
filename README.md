# Gamelist

Proyecto de fin de ciclo

## Descripción

Sitio web que permite la creación de listas de videojuegos para los usuarios registrados. Estas listas se dividen en cuatro categorías (Completado, En progreso, Pendiente de jugar, Abandonado).

## Instalación

Puedes probar el funcionamiento del sitio web utilizando el servidor de desarrollo de PHP utilizando el siguiente comando:

`php -S localhost:8000`
